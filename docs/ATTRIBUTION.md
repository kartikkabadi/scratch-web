# Attribution

Scratch Web is an experimental, independent, unofficial companion for
[Scratch](https://github.com/erictli/scratch) by erictli.

Scratch Web would not be possible without Scratch. Scratch defines the original
product, interaction model, notes format, and much of the behavior Scratch Web
aims to make available from mobile browsers.

## Upstream Reference

- Repository: https://github.com/erictli/scratch
- Author: erictli
- License observation: the upstream README states MIT.
- Standalone upstream LICENSE file: not present in the inspected clone.
- Reference commit: `6d443e9a6eb3e0742d84c2b5abdca46a0270f7c7`

## Project Relationship

Scratch Web is independent and unofficial unless that changes. The project is
structured respectfully so future upstream discussion or PR work is possible.

## Reuse Policy

When porting or copying substantial code from Scratch:

- Preserve source comments and notices.
- Link back to upstream.
- Record the upstream commit used.
- Keep attribution visible in README, docs, and About surfaces.
- Before copying substantial upstream source or publishing a public beta, verify
  the latest upstream repository/release license evidence and record the exact
  commit/date/source here.
- If a standalone upstream LICENSE file is still absent, document that risk and
  prefer behavioral reimplementation over large copied source until license
  evidence is clarified.
