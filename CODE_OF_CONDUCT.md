# Code Of Conduct

Scratch Web uses a simple contribution standard:

- Be respectful and specific.
- Assume good intent, but accept corrections quickly.
- Keep feedback about code, behavior, risk, and user impact.
- Do not share private notes, secrets, device identifiers, or private network
  details.
- Maintainers may remove comments or contributions that are abusive, spammy,
  unsafe, or unrelated to the project.

For security-sensitive issues, use the private reporting path in
[SECURITY.md](SECURITY.md).
